## Getting Started 

### Mac

`sudo pip install -U numpy scipy matplotlib pandas ipython `

### Ubuntu

`sudo apt-get install python-numpy python-scipy python-matplotlib ipython ipython-notebook python-pandas`

### Windows
- Use a virtual machine with ubuntu
- Or use Anaconda
